# Hey, I'm Kendall

I went from managing retail floors to managing React state, and the road between was anything but conventional. After six years leading teams in retail and nonprofits — including running STEM programs for underrepresented communities — I taught myself to code and never looked back. A decade later, I'm a Senior Software Engineer building fast, polished web experiences.
